# GLESExamples

## 项目简介
本项目是基于opengles3 开发的demo, 主要目的在于入门 Opengles 渲染。具体效果见B站：https://www.bilibili.com/video/BV1aY4y167Eh?spm_id_from=333.999.0.0

## 开发环境
· Android 10 + java 1.8 + ndk24\
· 测试机器: Redmi k30 Pro\
· glm
